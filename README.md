# LibrarySystem

The NEU Library System is a web-based application designed to manage and monitor library visits and user activities. The system allows students to register, log their library visits, and helps administrators track usage efficiently.

This project aims to improve the traditional manual logging process by providing a digital platform for recording and managing library data.

Objectives
- Provide a digital library logging system for students
- Monitor library visits in an organized database
- Improve efficiency in tracking library users
- Support administrators in managing library records

 Features
- Student Registration
- User Login System
- Library Visit Logging
- Time In and Time Out Recording
- User Blocking System
- Admin Monitoring Dashboard
- User Activity Tracking
- Technologies Used

HTML5 – Website structure
CSS3 / Bootstrap – User interface design
JavaScript – Client-side functionality
Firebase – Database and authentication services
